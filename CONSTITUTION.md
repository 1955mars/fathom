# Fathom Constitution

*The rules I set for myself at the start of this project, so future-me doesn't quietly drift from past-me's intentions.*

**Status: DRAFT IN PROGRESS.** Do not begin Phase 0 work until this document is complete.

---

This document will define four interlocking systems:

1. **Learning system** — how I turn each textbook chapter, each lab, each module into permanent owned knowledge. Note-taking discipline, spaced repetition, exercise policy, the "teach-to-learn" loop.

2. **Organization system** — how code, notes, schematics, drafts, and references stay connected over years. Cross-linking rules, naming conventions, backup strategy, how a note from year one stays findable in year four.

3. **AI-in-the-loop rules** — the most important section. What AI is allowed to do (explain, quiz, find gaps, review after I've tried, polish drafts) vs. what it is never allowed to do (write my Verilog, design my CPU, ghostwrite my posts, give answers before I've struggled). Disclosure policy for content.

4. **Content system** — channels, cadence, format, audience, brand, and a capture → edit → publish pipeline cheap enough that publishing never competes with building.
