## CONSTITUTION §1 drafted

Drafted the Learning system section of CONSTITUTION.md. Key decisions locked:

- **Anchor:** Feynman technique is the central learning mechanism.
- **Audience:** past self — EE/CS literate, but no topic knowledge. Expects rigor, motivation before mechanism, no unexplained jargon.
- **Concept granularity:** anything whiteboard-explainable in 2–10 minutes. Smaller folds into parent; larger splits into sub-concepts.
- **Per-concept artifacts:** written markdown doc in `notes/<source>/<concept>.md` + private unedited spoken recording in `content/videos/private/` (gitignored).
- **Public content** (blog posts, edited videos) is a separate, selective layer — not required for every concept.
- **Success criterion:** all three of (1) cold write, (2) fluid single-take speak, (3) next-day re-read still clear.
- **Scope:** every concept. No "feels simple, skip it."
- **Gate:** hard — no new chapter/module until current batch passes.
- **Gap handling:** hollow discoveries logged in `LOG/` for visibility over time.
- **Formalism:** allowed and often required (timing diagrams, waveforms, code), but intuition must come first.

## CONSTITUTION §2 drafted

Drafted Organization system. Key rules codified:

- **Tool:** Obsidian as lens; markdown remains the source of truth.
- **Vault:** whole Fathom repo is a single vault, `.obsidian/` at root.
- **Folder roles:** `notes/<source>/` holds source-bound notes (chapter summaries, exercises, running questions); `notes/concepts/` holds the canonical Feynman docs. Concepts graduate from source to concepts via `git mv`.
- **Filenames:** kebab-case slugs; first line is `# Pretty Title`.
- **Required cross-links:** concept → source + prerequisites + enables; LOG → concepts + commits touched that day; blog → concept docs it builds on.
- **Plugin whitelist:** Dataview, Excalidraw, Templater. Amendment required to add others.
- **`.obsidian/` policy:** commit only `community-plugins.json`; ignore the rest.
- **Tag taxonomy:** phase / domain / tool / status. New categories require an amendment; within categories, additions free.
- **Backup:** GitHub daily + Time Machine. No iCloud/Dropbox/OneDrive.
- **Forbidden patterns:** silent renames, orphan docs, committing private recordings, adding Obsidian plugins outside whitelist.
